# node-ts-starter

## Installation

``` TypeScript
$~ npm install
```

## Build

Build once:

``` TypeScript
$~ npm run build
```

or test and watch for changes: 

``` TypeScript
$~ npm run build:watch
```

## Test

Test once:

``` TypeScript
$~ npm test
```

or test and watch for changes: 

```
$~ npm run test:watch
```

## Run

``` TypeScript
$~ npm start
```
